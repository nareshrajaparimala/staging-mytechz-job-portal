11. EXPECTED USER FLOWS
═══════════════════════════════════════════════════════════════

FLOW 1: Job Seeker Browsing Jobs
1. Visit homepage
2. See navbar: Home | Jobs | Resume | Admissions | Webinars
3. Hover over "Jobs"
4. See dropdown: Private Jobs, Government Jobs
5. Click "Private Jobs" → Go to jobs listing
6. Click "Government Jobs" → See "Coming Soon" page

FLOW 2: Recruiter Accessing Dashboard
1. Go to login page
2. Select "Recruiter" role
3. Enter credentials and login
4. See navbar with "For Employers" added
5. Hover over "For Employers"
6. See dropdown: Buy Online, Talent Cloud, Employer Dashboard
7. Click "Employer Dashboard" → Go to recruiter dashboard

FLOW 3: Mobile User Navigation
1. Visit on mobile device
2. See hamburger menu icon
3. Click hamburger → Menu slides down smoothly
4. Icon rotates 90°
5. Click "Jobs" → Submenu expands with arrow rotation
6. Click "Private Jobs" → Navigate to jobs page
7. Menu closes automatically

FLOW 4: WhatsApp Contact
1. Browse any page
2. See green WhatsApp button at bottom-right
3. Button pulses to attract attention
4. Click button
5. WhatsApp opens with pre-filled message
6. Phone number: +91 8618659319

12. TESTING SCENARIOS
═══════════════════════════════════════════════════════════════

SCENARIO 1: Verify Role-Based Visibility
✓ Login as Job Seeker → "For Employers" NOT visible
✓ Login as Recruiter → "For Employers" IS visible
✓ Login as Admin → "For Employers" NOT visible
✓ Not logged in → "For Employers" NOT visible

SCENARIO 2: Test Dropdown Interactions
✓ Desktop: Hover shows dropdown
✓ Desktop: Move mouse away, dropdown hides
✓ Mobile: Click shows dropdown
✓ Mobile: Click again, dropdown hides
✓ Mobile: Arrow rotates correctly

SCENARIO 3: Test Navigation
✓ Click "Private Jobs" → Goes to /jobs/private
✓ Click "Government Jobs" → Goes to /jobs/government
✓ Click "Buy Online" → Goes to /recruiter/subscription
✓ Click "Talent Cloud" → Goes to /recruiter/resume-database
✓ Click "Employer Dashboard" → Goes to /recruiter

SCENARIO 4: Test Animations
✓ Hamburger menu slides smoothly
✓ Icon rotates 90° when opening
✓ Backdrop blur appears
✓ Arrow rotates 180° in dropdowns
✓ All transitions are smooth (0.3s)

SCENARIO 5: Test Responsiveness
✓ Desktop: All items visible, hover works
✓ Tablet: Some items in hamburger, layout adapts
✓ Mobile: All items in hamburger, touch-friendly
✓ Resize window: Layout adapts smoothly
