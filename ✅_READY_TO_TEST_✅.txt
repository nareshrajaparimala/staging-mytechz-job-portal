🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

ISSUE: For Employers not showing
FIX: 
- Make sure you're logged in as recruiter
- Check browser console for errors (F12)
- Clear cache and hard refresh (Ctrl+F5)
- Verify userInfo.role === 'recruiter'

ISSUE: Dropdown not appearing on hover
FIX:
- Check if you're on desktop view (not mobile)
- Clear browser cache
- Check z-index in CSS (should be 10000)
- Verify hover styles are applied

ISSUE: Mobile menu not animating
FIX:
- Check if mobile view is active (< 767px)
- Clear browser cache
- Check console for JavaScript errors
- Verify mobileMenuOpen state is toggling

ISSUE: WhatsApp button not visible
FIX:
- Check if WhatsAppFloat component is imported in App.jsx
- Verify z-index (should be 9999)
- Check if button is behind other elements
- Clear cache and refresh

ISSUE: Changes not reflecting
FIX:
- Save all files
- Wait for Vite hot reload (check terminal)
- Hard refresh browser (Ctrl+F5)
- Clear browser cache completely
